# 半导体+新能源整理文件夹

本文件夹用于把“半导体相关分析及学习材料”和“新能源方向学习与分析对话需求”合并到同一套可继续迭代的研究资料中。

## 文件结构

| 路径 | 内容 | 用途 |
|---|---|---|
| `01_半导体学习与分析整理.md` | 半导体产业学习主线、光存芯算、A股验证框架 | 学习半导体产业链与A股映射 |
| `02_新能源对话整理.md` | 新能源方向学习需求、产业链拆解、复盘方法 | 延续新能源手册对话的整理版 |
| `03_半导体+新能源联合复盘框架.md` | 两大方向的共通研究方法、轮动和数据验证 | 日常复盘、主题比较、组合观察 |
| `04_资料索引.md` | 当前文件夹内全部整理材料的索引 | 快速定位源材料、图片、数据和工具 |
| `data/memory_hbm_20260629_pool.csv` | HBM/存储相关股票池 | 半导体存储方向样本池 |
| `data/stock_pool.csv` | 当前仓库股票筛选池副本 | 技术筛选工具的基础股票池 |
| `images/` | 半导体、新能源、算力、产业链相关图片 | 后续写作和复盘配图参考 |
| `source/` | 原始粘贴文本和来源材料 | 后续追溯、重写、扩展 |
| `source/current_repo/新能源/` | 当前仓库 `新能源/` 文件夹副本 | 保留新能源手册、Word、脚本和图片 |
| `source/current_repo/docs/` | 当前仓库 `docs/` 文件夹副本 | 保留半导体学习 Word、101楼学习整理、交易纪律和参考文档 |
| `tools/stock_screen.py` | 股票技术筛选脚本副本 | 后续按半导体/新能源股票池筛选 |

## 使用顺序

1. 先读 `01_半导体学习与分析整理.md`，理解半导体产业链和“光、存、芯、算”。
2. 再读 `02_新能源对话整理.md`，理解新能源从资源、材料、设备、应用到运营的传导。
3. 最后读 `03_半导体+新能源联合复盘框架.md`，把两个方向放到同一个A股复盘框架中比较。
4. 需要找原始材料、图片和数据时，读 `04_资料索引.md`。
