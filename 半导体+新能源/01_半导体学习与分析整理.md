# A股半导体学习与分析整理

> 来源：`source/半导体学习文档需求_初版.txt`、`source/半导体学习文档升级需求.txt`、`source/核心科技.txt`、`data/memory_hbm_20260629_pool.csv`。  
> 定位：面向股票小白的半导体产业链学习与A股研究框架。  
> 提醒：本文不构成投资建议，所有实时数据均需更新验证。

## 1. 半导体学习主线

学习半导体不能从股票名单开始，而要先理解“一颗芯片如何诞生”。半导体产业链可以简化成：

```text
芯片设计 -> EDA/IP -> 晶圆制造 -> 半导体设备/材料 -> 封装测试 -> 模组/板卡 -> 终端应用
```

小白容易把“芯片”理解成一个单品，其实A股公司可能只在其中一个环节有业务：有的做设计，有的做晶圆制造，有的做设备，有的做材料，有的做封测，有的只是下游模组或服务器配套。研究时必须先问：这家公司到底在产业链哪一层？

## 2. 关键概念随文解释

| 概念 | 小白解释 | A股研究怎么看 |
|---|---|---|
| 半导体 | 导电能力介于导体和绝缘体之间的材料，芯片主要基于半导体材料制造 | 半导体不是单一股票概念，要拆成设计、制造、设备、材料、封测 |
| 芯片 | 把电路集成到硅片上的产品，是电子设备的大脑或器官 | 看芯片类型、客户、应用场景、国产替代程度 |
| 集成电路 | 把大量晶体管和电路集成在一个芯片上 | 关注设计能力、制造工艺、封装能力 |
| 晶圆 | 圆形硅片，一整片晶圆上同时制造很多颗芯片 | 晶圆厂产能、良率、制程决定供应能力 |
| 制程 | 芯片制造工艺节点，常见如先进制程、成熟制程 | 先进制程看高端算力，成熟制程看汽车、工业、模拟等 |
| Fabless | 只做芯片设计，不自建晶圆厂 | 看设计能力、客户、流片成功率 |
| Foundry | 晶圆代工厂，帮设计公司制造芯片 | 看产能利用率、制程、客户结构 |
| IDM | 设计、制造、封测一体化公司 | 看全链条能力和资本开支 |
| OSAT | 封装测试服务商 | 看先进封装、客户、稼动率 |
| EDA | 芯片设计软件 | 国产替代关键环节，但收入和客户导入需查公告确认 |
| IP核 | 可复用的芯片设计模块 | 看授权、客户和生态 |
| 流片 | 把设计交给晶圆厂试生产 | 流片成功不等于量产盈利 |
| 良率 | 合格芯片比例 | 良率提升直接影响成本和利润 |

## 3. 半导体产业链总览

| 环节 | 做什么 | 关键产品/技术 | 技术壁垒 | A股研究重点 |
|---|---|---|---|---|
| 芯片设计 | 设计芯片架构和电路 | CPU、GPU、MCU、模拟、存储控制 | 架构、算法、客户 | 客户导入、收入占比、流片风险 |
| EDA/IP | 提供设计工具和模块 | EDA软件、IP核 | 软件生态、验证能力 | 国产替代进度、客户质量 |
| 晶圆制造 | 把设计做成晶圆上的电路 | 光刻、刻蚀、沉积、离子注入 | 工艺积累、良率、资本开支 | 产能利用率、制程、扩产 |
| 设备 | 晶圆厂生产工具 | 光刻、刻蚀、薄膜、清洗、CMP、检测 | 技术和客户验证周期 | 国产替代、订单、验收 |
| 材料 | 制造过程耗材 | 硅片、光刻胶、特气、靶材、CMP材料 | 纯度、稳定性、认证周期 | 客户认证、收入占比 |
| 封测 | 把裸芯片封装并测试 | 传统封装、先进封装、Chiplet、CoWoS | 工艺、良率、客户 | 先进封装订单、稼动率 |
| 模组/板卡 | 把芯片组合成可用产品 | 存储模组、PCB、服务器板卡 | 客户、供应链 | AI服务器、AIDC需求 |

## 4. 光、存、芯、算

半导体学习材料里反复强调“光、存、芯、算”，它可以理解为AI硬件链的四个入口。

| 方向 | 含义 | 产业逻辑 | 关键验证 |
|---|---|---|---|
| 光 | 光通信、光模块、硅光、CPO | AI集群内部需要高速互联，带动光模块和光器件 | 订单、客户、速率升级、毛利率，均需更新验证 |
| 存 | DRAM、NAND、HBM、SSD、存储模组 | AI训练和推理需要高带宽存储，HBM成为算力配套重点 | 存储价格、HBM配套、封装测试、客户导入，均需更新验证 |
| 芯 | 国产CPU/GPU/AI芯片、EDA、设备材料 | 国产替代和自主可控主线 | 流片、客户、收入占比、公告确认 |
| 算 | AI服务器、AIDC、液冷、电源、PCB、连接器 | 算力基础设施落地 | 服务器订单、数据中心建设、液冷渗透率，均需更新验证 |

## 7. 半导体研究验证框架

| 信息来源 | 证据等级 | 怎么用 |
|---|---|---|
| 公告、年报、季报、问询回复 | 高 | 确认业务、客户、订单、收入占比 |
| 招投标、中标、客户公告 | 高 | 验证订单和客户导入 |
| 行业数据、协会数据 | 中 | 判断周期、价格、需求 |
| 投资者关系、互动平台 | 中低 | 只能辅助，需回到公告财报 |
| 论坛、小作文、图片观点 | 低 | 只能作为待验证假设 |

## 8. 常见误区

1. 看到“国产替代”就认为公司受益：必须验证公司是否在核心环节、收入占比多少、客户是否导入。
2. 看到“先进封装”就认为业绩爆发：要查先进封装订单、产能、稼动率和毛利率。
3. 看到“HBM”就买存储概念：要区分存储芯片、接口芯片、封测、设备、材料和载板。
4. 把小作文当事实：必须拆成公告、财报、订单、客户、价格、产能验证项。
