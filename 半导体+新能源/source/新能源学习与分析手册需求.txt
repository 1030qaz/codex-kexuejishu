### 4. 新能源行情常见的轮动路径

请不要只使用图片中的单一路径，而是总结多种可能路径。

至少包括：

#### 路径一：资源涨价型

上游资源 → 中游材料 → 电池厂 → 下游应用 → 设备补涨

适用场景：
- 锂价、镍价、钴价、硅料价格等上行；
- 供需紧张；
- 库存低；
- 需求恢复。

#### 路径二：需求恢复型

新能源车销量 / 储能招标 / 光伏装机 → 电池与系统 → 中游材料 → 上游资源 → 设备

适用场景：
- 终端需求先改善；
- 下游排产增加；
- 材料企业订单恢复；
- 上游价格企稳。

#### 路径三：供给出清型

亏损企业减产 → 产品价格企稳 → 龙头毛利率修复 → 估值修复 → 小票弹性补涨

适用场景：
- 行业经历长期价格战；
- 产能过剩后开始出清；
- 低位板块出现反转预期。

#### 路径四：技术迭代型

新技术突破 → 设备 / 材料先涨 → 中试验证 → 客户认证 → 量产公司兑现

适用场景：
- 固态电池；
- 钠电池；
- 复合铜箔；
- 钙钛矿；
- HJT；
- BC电池；
- 氢能电解槽；
- 新型储能。

#### 路径五：政策催化型

政策预期 → 主题扩散 → 龙头确认 → 订单验证 → 业绩兑现 → 分化退潮

适用场景：
- 储能政策；
- 电力改革；
- 新能源车补贴；
- 充电桩政策；
- 风光大基地；
- 氢能规划；
- 新型电力系统。

#### 路径六：业绩预增型

行业数据改善 → 公司订单改善 → 中报预增预期 → 业绩预告 → 机构重估

适用场景：
- 中报、三季报、年报窗口期；
- 公司低位但业绩开始改善；
- 市场尚未充分定价。

## 第八章：中报预增、预期差和小作文验证方法

请专门写一章，适合A股实战复盘。

### 1. 如何筛选中报预增方向

说明：

- 哪些行业数据领先于财报；
- 哪些价格变化会提前反映到业绩；
- 哪些公司可能有低基数；
- 哪些公司可能有毛利率修复；
- 哪些公司可能有库存减值压力；
- 哪些公司可能表面增长但现金流差。

### 2. 如何寻找预期差

说明：

- 市场认为行业差，但公司业绩实际改善；
- 板块低位，但细分方向订单变好；
- 原材料价格企稳后，中游利润修复；
- 大公司没弹性，小公司有业绩拐点；
- 公司主业被忽视，新业务被验证；
- 传统公司切入新能源，但需验证收入占比。

### 3. 如何处理小作文

将小作文分为：

- 纯概念型；
- 订单型；
- 价格上涨型；
- 技术突破型；
- 客户认证型；
- 中报预增型；
- 政策催化型；
- 出海型；
- 并购重组型。

每类都要说明：

- 该看什么公告；
- 该查什么数据；
- 该问什么问题；
- 如何判断真假；
- 如何判断是否已被股价反映。

### 4. 如何分析澄清公告

请按照以下分类分析：

1. 单纯澄清没有相关业务；
2. 澄清没有某项热门业务，但原有业务增长较快；
3. 有相关业务但收入占比很小；
4. 有相关业务且有订单、产能、客户认证；
5. 有技术储备但尚未量产；
6. 有合作协议但没有实质收入。

每种情况说明：

- 是否还能继续观察；
- 需要验证哪些数据；
- 什么时候说明逻辑证伪；
- 什么时候说明预期差仍在。

## 第十章：新能源行情中的常见误区

请详细说明小白容易犯的错误。

包括：

1. 只看概念，不看业绩；
2. 只看股价涨了，不看产业链位置；
3. 把上游涨价误认为全产业链利好；
4. 把原材料下跌误认为全产业链利空；
5. 把小作文当作确定性；
6. 不区分动力电池和储能电池；
7. 不区分正极、负极、隔膜、电解液；
8. 不理解顺价；
9. 不看库存和价格；
10. 不看公司公告澄清；
11. 不看财务报表中的存货跌价和现金流；
12. 不区分光伏主链和辅材；
13. 不理解光伏产能过剩；
14. 不理解风电招标和并网节奏；
15. 不理解储能招标量和盈利质量的区别；
16. 在行情后期追高后排小票；
17. 把长期好行业等同于短期好股票；
18. 把政策空间等同于公司业绩；
19. 把订单公告等同于利润兑现；
20. 忽视减持、定增、解禁、商誉和资产减值风险。
