4. 若黑话和原文语境冲突，以原文上下文为准。

## 0621 更新：存储扩产与半导体材料链参考

### 总逻辑

随着全球 AI 算力需求爆发，存储芯片成为产业核心。SK 海力士宣布 2030 年前在韩国龙仁投资超 150 亿美元扩产 DRAM 产能，目标翻倍至 100 万片/月，并加速布局 HBM；国内存储龙头长鑫科技亦启动大规模扩产，计划年内建成新厂并新建两座工厂，总产能翻倍，冲击全球前三。

两大巨头的扩产计划，直接拉动上游半导体材料需求；叠加国产替代政策驱动，抛光液、前驱体、硅片、靶材、光刻胶、电子特气等核心材料赛道进入重点观察期。

分析时应重点拆成：

1. AI 算力需求 -> HBM/DRAM 扩产。
2. 存储扩产 -> 晶圆制造产能提升。
3. 晶圆制造产能提升 -> 上游核心材料耗材需求提升。
4. 海外龙头扩产 + 国内长鑫扩产 -> 国产替代逻辑强化。
5. 个股分析必须回到材料类别、国产化率、技术壁垒、客户绑定和产能释放节奏。

### 材料链拆分

| 方向 | 核心逻辑 | 参考标的 |
|---|---|---|
| 抛光材料（CMP） | 抛光材料（抛光液/垫）是芯片制造中 CMP 工艺的核心耗材，占比约 7%。长鑫/海力士扩产推动 12 英寸硅片及先进制程产能提升，直接拉动抛光材料需求。国产替代率约 20%，技术壁垒高但已有突破。 | 鼎龙股份、安集科技 |
| 前驱体 | 前驱体是芯片薄膜沉积工艺的关键材料，技术壁垒极高，纯度要求 6N 以上。长鑫扩产推动前驱体需求激增，国产化率约 30%，替代空间较大。 | 雅克科技 |
| 硅片 | 硅片占芯片材料成本约 40%，12 英寸硅片是长鑫/海力士扩产核心耗材。国产化率不足 20%，大硅片国产化提速。 | 沪硅产业 |
| 靶材 | 靶材是物理气相沉积（PVD）关键材料，3nm 及 HBM 需求拉动高纯靶材（7N 级）需求。国产化率约 15%-20%，高端制程国产突破是关键。 | 江丰电子 |
| 光刻胶 | 光刻胶是光刻工艺核心，ArF 和 EUV 胶国产化率不足 5%，技术壁垒最高。长鑫/海力士扩产推动先进制程光刻胶需求。 | 彤程新材、南大光电 |
| 电子特气 | 电子特气占芯片制造成本约 13%，高纯度特气（6N 级）被日企垄断较多，国产化率约 10%-15%。扩产推动需求提升，国产替代迫切。 | 华特气体、中船特气 |
| 主板材料/化工补充观察 | 主板方向可结合材料属性、客户验证、产能节奏和估值位置观察。 | 厦门钨业、昊华科技 |

### 后续分析使用方式

当发言出现“长鑫扩产”“海力士扩产”“HBM”“DRAM”“存储”“材料”“大硅片”“CMP”“前驱体”“靶材”“光刻胶”“电子特气”等关键词时，分析应优先检查：

- 这条发言是在讲存储主线、材料补涨，还是国产替代分支？
- 是直接受益于存储扩产，还是二阶/三阶映射？
- 标的是耗材、设备、材料、封测，还是设计端？
- 当前逻辑是供需改善、国产替代、客户验证、产能释放，还是情绪估值上修？
- 是否已经从产业逻辑切换到交易拥挤和估值兑现阶段？

输出建议：

```text
产业链还原：AI算力需求 -> 存储/HBM扩产 -> 晶圆产能提升 -> 上游材料需求 -> 对应材料类别/标的
```
