# A股新能源方向学习与分析手册

> 版本：2026-06-25 深度重写版  
> 定位：面向股票小白的新能源产业链学习、研究、验证和复盘手册。  
> 边界：本文不提供买入、卖出或持有建议。所有实时行情、估值、价格、订单、销量、装机、排产、财务同比、市场份额、客户份额，均必须标注“需更新验证”。

## 使用说明：先学会拆逻辑，再学会看股票

很多人第一次看新能源，容易把它理解成“锂电池”“光伏”或者“新能源车”。这会导致两个问题：第一，只看到热门词，看不到产业链位置；第二，只看到股票涨跌，看不到涨跌背后是价格、订单、业绩、政策还是情绪。

本手册按一个固定学习顺序展开：

1. 概念解释：先用小白能听懂的话说明这个方向是什么。
2. 产业链位置：它在上游、中游、下游、设备端还是运营端。
3. 盈利模式：公司到底靠什么赚钱，利润弹性来自哪里。
4. 数据验证：哪些数据能证明逻辑正在发生，哪些数据会证伪逻辑。
5. 公司案例：用重点A股公司说明同一个行业里不同环节的差异。
6. 小白误区：告诉你最容易误判的地方。

论坛发言、图片观点、小作文、会议纪要、群聊传闻，一律先当作“待验证假设”。正确做法不是马上下结论，而是把它改写成可验证问题：它说的是涨价、订单、产能、客户、技术、政策，还是业绩？对应要查公告、财报、行业价格、招投标、排产、客户认证还是澄清公告？

## 第一章：新能源行业总览

### 1. 新能源到底是什么

新能源不是一个单一行业，而是一套围绕“更清洁的发电、更高效的用电、更灵活的储电、更智能的电网”的产业系统。它包括锂电池、新能源车、储能、光伏、风电、氢能、电力设备、新型电力系统、绿电运营，也包括AI算力用电、数据中心储能、低空经济电池、机器人电池等交叉方向。

和传统能源相比，新能源有三个特点：

| 特点 | 小白解释 | 对A股研究的意义 |
|---|---|---|
| 渗透率提升 | 越来越多车、电站、工厂、数据中心使用新能源体系 | 容易形成长期成长预期 |
| 成本下降 | 技术进步和规模制造让单位成本下降 | 容易刺激需求，但也可能导致价格战 |
| 周期波动 | 企业扩产太快时会产能过剩，价格下跌 | 新能源既像成长股，也像周期股 |

新能源的大行情通常来自五类共振：政策催化、产业趋势、价格周期、业绩兑现、市场情绪。只看其中一个因素都不够。例如，政策支持储能，不代表每个储能公司都赚钱；锂价上涨，不代表电池厂和整车厂也受益；光伏长期空间大，不代表短期组件企业一定有利润。

### 2. 新能源产业链全景

| 层级 | 包含内容 | 研究重点 |
|---|---|---|
| 上游资源 | 锂、镍、钴、磷、石墨、硅、铜、铝、稀土、氟化工 | 价格、库存、成本曲线、资源自给率 |
| 中游材料 | 正极、负极、隔膜、电解液、铜箔、铝箔、胶膜、玻璃、银浆、石英材料 | 加工费、开工率、顺价能力、良率 |
| 制造端 | 电芯、PACK、组件、风机、逆变器、PCS、变压器、电解槽 | 出货量、客户结构、单位盈利 |
| 设备端 | 锂电设备、光伏设备、风电设备、氢能设备、电网设备、检测设备 | 订单、验收、回款、资本开支 |
| 应用端 | 新能源车、储能、光伏电站、风电场、数据中心、电网、工商业用户 | 销量、装机、招投标、利用小时 |
| 运营端 | 绿电运营、储能运营、虚拟电厂、电力交易、辅助服务 | 电价、收益率、利用率、政策机制 |

看新能源股票，第一件事是判断公司在这张图里的位置。上游资源涨价弹性大，但跌价风险也大；中游材料容易内卷，关键看顺价和产能利用率；设备端看订单周期；运营端看资产收益率和电价机制；电力设备看电网投资和招标节奏。

### 3. 新能源常见行情来源

| 行情来源 | 它通常怎么发生 | 要验证什么 |
|---|---|---|
| 政策催化 | 补贴、规划、标准、地方项目、示范工程发布 | 是否有预算、时间表和强制约束 |
| 价格反转 | 锂盐、硅料、组件、储能中标价止跌 | 是否连续、是否带来毛利率修复 |
| 需求恢复 | 车销量、储能招标、光伏装机、风电招标改善 | 是否传导到订单、排产和收入 |
| 供给出清 | 亏损企业减产，落后产能退出 | 龙头毛利率是否改善 |
| 技术迭代 | 固态电池、钠电、HJT、BC、钙钛矿、电解槽 | 是否有中试、认证、量产、客户 |
| 业绩预增 | 中报、三季报、年报窗口出现改善预期 | 是主业改善还是一次性收益 |
| 估值修复 | 板块长期下跌后低位放量 | 是否有基本面配合 |
| 主题扩散 | 小作文密集、连板、龙虎榜活跃 | 是否只是情绪，必须待验证 |

## 第二章：小白必须掌握的分析语言

### 1. 上游、中游、下游

上游是资源和原材料，中游是材料与制造，下游是应用和终端客户。这个划分很重要，因为同一条产业链里，价格上涨对不同环节的影响可能完全相反。

以锂电池为例：锂价上涨时，锂矿公司可能利润改善；正极材料企业如果不能顺价，毛利率可能被压；电池厂和整车厂如果不能把成本转嫁给消费者，利润可能下降。所以“小白误区”之一就是把上游涨价简单理解为全产业链利好。

### 2. 顺价、加工费和毛利率

顺价是指原材料涨价后，公司能不能把成本涨幅转嫁给客户。加工费是中游企业在原料价格之外赚到的制造差价。毛利率是收入扣掉直接成本后剩下的比例。

| 概念 | 小白解释 | 看什么数据 |
|---|---|---|
| 顺价 | 成本涨了，公司能不能跟客户涨价 | 毛利率、合同条款、客户结构 |
| 加工费 | 不赚资源涨价的钱，赚加工制造的钱 | 行业报价、单位毛利 |
| 毛利率 | 产品卖出去后直接赚多少 | 财报毛利率，需更新验证 |
| 存货跌价 | 库存买贵了，价格跌后要计提损失 | 财报存货和减值，需更新验证 |
| 合同负债 | 客户预付款，可能说明订单景气 | 财报合同负债，需更新验证 |
| 经营现金流 | 公司利润有没有变成现金 | 现金流量表，需更新验证 |

### 3. 成长股、周期股、主题股

新能源经常同时具有三种属性：

| 属性 | 代表逻辑 | 典型验证 |
|---|---|---|
| 成长 | 渗透率提升、技术替代、出海扩张 | 出货量、客户、收入增长，均需更新验证 |
| 周期 | 价格涨跌、产能过剩、库存变化 | 价格、库存、开工率，均需更新验证 |
| 主题 | 政策、会议、小作文、跨界概念 | 公告、订单、收入占比，均需更新验证 |

一个好研究问题不是“这是不是新能源概念”，而是“当前市场炒的是成长、周期反转、业绩预增、估值修复，还是纯题材补涨”。不同逻辑的验证方式完全不同。

## 第三章：锂电池产业链

### 1. 概念解释

锂电池是一种可反复充放电的电池，核心是锂离子在正极和负极之间移动。它的主要应用包括新能源汽车动力电池、储能电池和消费电子电池。

一块锂电池背后不是一个公司完成的，而是一条长链条：锂矿和锂盐提供资源，正极、负极、隔膜、电解液等材料决定性能和成本，电芯厂负责制造，PACK和BMS负责电池包安全管理，最后卖给车企、储能系统商或消费电子客户。

### 2. 产业链位置

| 环节 | 代表产品 | 产业链位置 | 核心矛盾 |
|---|---|---|---|
| 锂资源 | 锂矿、盐湖、锂精矿 | 上游资源 | 锂价、成本、权益产量 |
| 锂盐 | 碳酸锂、氢氧化锂 | 上游加工 | 原料成本和锂盐价格 |
| 正极 | 磷酸铁锂、三元、高镍、钠电正极 | 中游材料 | 加工费、路线、客户 |
| 负极 | 人造石墨、天然石墨、硅基负极 | 中游材料 | 石墨化成本和快充路线 |
| 隔膜 | 湿法、干法、涂覆膜 | 中游材料 | 良率、涂覆、客户 |
| 电解液 | 电解液、六氟、LiFSI、添加剂 | 中游材料 | 配方和原料价格 |
| 铜箔铝箔 | 集流体、复合铜箔 | 中游材料 | 加工费、薄型化、量产 |
| 电芯PACK | 动力电池、储能电池、消费电池 | 制造端 | 出货、良率、单Wh盈利 |

### 3. 盈利模式

锂电链的利润在不同周期阶段会重新分配。锂价上涨时，上游资源利润弹性通常最大；锂价稳定时，中游材料和电池厂可能修复利润；锂价快速下跌时，资源股承压，中游可能面临库存跌价，但下游成本压力可能减轻。

| 情景 | 更可能受益 | 更可能承压 | 关键问题 |
|---|---|---|---|
| 锂价上涨 | 低成本锂矿、锂盐 | 无法顺价的材料和电池厂 | 下游能否接受涨价 |
| 锂价企稳 | 材料、电芯、储能系统 | 纯资源弹性下降 | 毛利率是否修复 |
| 锂价下跌 | 电池厂、整车、储能需求 | 高成本资源、库存高企业 | 是否出现存货减值 |

### 4. 行情轮动路径

锂电行情常见八阶段：

1. 锂价触底：价格止跌、库存下降，市场开始寻找左侧信号。
2. 上游资源反弹：锂矿锂盐先涨，因为利润弹性最直接。
3. 材料排产恢复：正极、负极、电解液开工率提升。
4. 电池厂利润修复：原料稳定后，单Wh盈利改善。
5. 储能和整车需求确认：招标、装机、销量验证需求。
6. 设备订单修复：电池厂重新扩产或技改。
7. 小票扩散补涨：低位材料、设备、概念股活跃。
8. 分化退潮：涨幅过大但业绩未兑现的公司回落。

### 5. 关键验证数据

| 数据 | 用来验证什么 | 注意事项 |
|---|---|---|
| 碳酸锂、氢氧化锂价格 | 上游价格周期 | 需更新验证 |
| 锂精矿价格和库存 | 资源端成本与供需 | 需更新验证 |
| 正极、负极、电解液报价 | 中游顺价和加工费 | 需更新验证 |
| 电池排产 | 下游需求是否恢复 | 需更新验证 |
| 动力电池装机量 | 动力需求 | 需更新验证 |
| 储能电池出货 | 储能需求 | 需更新验证 |
| 公司毛利率和存货 | 利润是否真实改善 | 需更新验证 |

### 6. 公司案例

宁德时代（300750）是电芯和系统端代表，核心看动力电池、储能电池、客户结构、单Wh盈利和海外业务，所有份额、盈利和订单数据均需更新验证。它的优势是规模、研发、客户和供应链管理；风险是价格战、客户议价、海外政策和技术路线变化。

赣锋锂业（002460）和天齐锂业（002466）代表锂资源与锂盐弹性，研究时要看权益资源、锂价、成本曲线、库存和资产减值，实时价格和产量均需更新验证。它们不是简单的“新能源成长股”，更像带成长空间的资源周期股。

湖南裕能（301358）、德方纳米（300769）代表磷酸铁锂正极，重点看正极加工费、客户结构、产能利用率和库存跌价。锂价稳定后，如果加工费和排产改善，正极可能进入利润修复；如果产能继续过剩，股价反弹可能只是估值修复或主题补涨。

恩捷股份（002812）代表隔膜，研究关键是单平米盈利、涂覆膜占比、客户份额和新产能消化，相关数据均需更新验证。隔膜看起来不像资源股那么刺激，但如果格局好、客户稳定，盈利质量可能更受关注。

### 7. 小白误区

| 误区 | 为什么错 | 正确做法 |
|---|---|---|
| 锂价涨就是全链利好 | 材料和电池厂可能被成本挤压 | 分环节看利润传导 |
| 电池装机增长就等于电池厂赚钱 | 价格战可能让量增利不增 | 看单Wh盈利和毛利率 |
| 小作文说固态电池就马上兑现 | 技术到量产有很长链条 | 查中试、认证、订单和收入 |
| 只看收入增长 | 存货减值和现金流可能恶化 | 看存货、应收和经营现金流 |

## 第四章：储能产业链

### 1. 概念解释

储能就是把电存起来，在需要的时候再放出来。风电和光伏发电不稳定，白天、夜晚、晴天、阴天、风大、风小都会影响发电，所以电力系统需要储能来削峰填谷、调频、备用和提升新能源消纳。

储能分为大储、工商业储能、户用储能和数据中心储能。大储主要服务电网和发电侧；工商业储能看峰谷价差；户储看海外居民电价和渠道；数据中心储能看AI算力用电和供电可靠性。

### 2. 产业链位置

| 环节 | 角色 | 主要产品 |
|---|---|---|
| 电芯 | 储能系统核心成本 | 磷酸铁锂储能电芯 |
| 系统集成 | 把电芯、PCS、BMS、EMS、温控消防集成 | 储能柜、集装箱储能 |
| PCS | 直流和交流之间转换 | 储能变流器 |
| BMS | 管理电池安全 | 电池管理系统 |
| EMS | 调度系统 | 能源管理系统 |
| 温控消防 | 安全保障 | 液冷、风冷、消防系统 |
| 运营 | 通过电价机制赚钱 | 独立储能、虚拟电厂 |

### 3. 盈利模式

储能不是“招标量越大越赚钱”。如果中标价太低，系统集成商可能收入增加但毛利率下降。储能研究的核心是：项目收益机制是否清晰，中标价格是否企稳，电芯成本下降是否能转化为毛利率改善，海外订单是否真实交付并回款。

| 环节 | 利润来源 | 风险 |
|---|---|---|
| 储能电芯 | 规模、寿命、安全、成本 | 价格战、安全事故 |
| 系统集成 | 方案能力、采购成本、交付 | 低价中标、回款慢 |
| PCS/逆变器 | 功率器件、认证、渠道 | 海外库存、竞争加剧 |
| 运营 | 峰谷价差、辅助服务、容量收益 | 政策不确定、利用率不足 |

### 4. 行情轮动路径

储能行情常见路径是：政策或电价机制预期 -> 招标量增加 -> 中标价格企稳 -> 电芯和PCS订单改善 -> 系统集成毛利率验证 -> 财报兑现 -> 运营和虚拟电厂扩散。

如果只停留在“储能招标多”，但中标价持续下行、毛利率没有改善、经营现金流恶化，这种行情更可能是主题催化而不是业绩驱动。

### 5. 关键验证数据

| 数据 | 验证问题 | 备注 |
|---|---|---|
| 储能招标量 | 需求是否增长 | 需更新验证 |
| 储能中标价 | 是否恶性低价竞争 | 需更新验证 |
| 电芯价格 | 系统成本是否下降 | 需更新验证 |
| 公司订单和合同负债 | 是否转化为收入 | 需更新验证 |
| 毛利率和现金流 | 是否真的赚钱 | 需更新验证 |
| 海外库存和渠道 | 出海是否健康 | 需更新验证 |

### 6. 公司案例

阳光电源（300274）同时涉及光伏逆变器、储能系统和PCS，研究重点是海外渠道、储能占比、毛利率和订单交付，相关数据均需更新验证。它的受益情景是海外储能需求、逆变器需求和电网侧储能建设；证伪条件是海外库存高企、毛利率下滑或储能订单无法兑现。

派能科技（688063）偏户储和储能电池系统，研究重点是海外户储库存、渠道恢复、产品毛利率和客户结构，相关数据均需更新验证。若海外库存仍高、收入未恢复，低位反弹可能只是估值修复。

盛弘股份（300693）、科士达（002518）、上能电气（300827）偏PCS和电力电子，研究重点是储能招标、海外认证、订单和毛利率。PCS不像电芯那样直接受锂价影响，但受功率器件、认证和渠道影响较大。

### 7. 小白误区

| 误区 | 正确理解 |
|---|---|
| 储能招标多就一定赚钱 | 要看中标价、毛利率和回款 |
| 储能电芯和储能系统是一回事 | 电芯赚制造钱，系统集成赚方案和交付钱 |
| 所有储能都受益于电价改革 | 不同地区峰谷价差和收益机制差异很大 |
| 户储出海永远高增长 | 海外库存、渠道和政策会造成周期波动 |

## 第五章：光伏产业链

### 1. 概念解释

光伏就是把太阳光转化为电。光伏产业链包括硅料、硅片、电池片、组件、辅材、逆变器和电站运营。它的长期逻辑是度电成本下降带来装机增长，短期矛盾是产能扩张太快导致价格战和利润下滑。

### 2. 产业链位置

| 环节 | 作用 | 研究重点 |
|---|---|---|
| 硅料 | 光伏主链上游原料 | 价格、库存、产能出清 |
| 硅片 | 把硅料切成片 | 尺寸、薄片化、开工率 |
| 电池片 | 把光能转电能 | TOPCon、HJT、BC、效率 |
| 组件 | 最终发电板 | 出货、价格、海外渠道 |
| 辅材 | 玻璃、胶膜、银浆、石英砂、坩埚 | 装机、耗材、格局 |
| 逆变器 | 直流变交流 | 海外认证、渠道、储能联动 |
| 电站 | 投资和运营资产 | 电价、利用小时、收益率 |

### 3. 盈利模式

光伏主链企业通常看规模、成本和技术路线。辅材企业看装机量和竞争格局。逆变器企业看海外渠道、认证和产品结构。电站运营企业看电价、利用小时和融资成本。

光伏最容易出现“小白误判”：组件价格下降会伤害组件厂利润，但也可能刺激下游装机；硅料价格下跌会压制硅料企业利润，但可能改善电池片和组件成本。必须分环节分析。

### 4. 行情轮动路径

光伏行情通常有两类路径：

第一类是出清修复路径：价格战 -> 全链亏损 -> 落后产能停产 -> 主链价格企稳 -> 龙头毛利率改善 -> 辅材和设备分化 -> 技术路线扩散。

第二类是技术迭代路径：TOPCon/HJT/BC/钙钛矿预期 -> 设备和材料先涨 -> 产线验证 -> 客户订单 -> 量产利润兑现 -> 分化。

### 5. 关键验证数据

| 数据 | 验证什么 |
|---|---|
| 硅料、硅片、电池片、组件价格 | 价格战是否缓和，需更新验证 |
| 光伏新增装机 | 需求是否增长，需更新验证 |
| 组件出口 | 海外需求和贸易风险，需更新验证 |
| 企业开工率 | 产能利用率是否恢复，需更新验证 |
| 毛利率和减值 | 利润是否真实改善，需更新验证 |
| TOPCon/HJT/BC订单 | 技术路线是否兑现，需更新验证 |

### 6. 公司案例

隆基绿能（601012）、晶科能源（688223）、晶澳科技（002459）、天合光能（688599）代表组件和电池片龙头，研究重点是价格、出货、海外销售、技术路线和毛利率，均需更新验证。它们的受益情景是价格企稳、装机恢复、技术路线领先；证伪条件是价格继续下跌、毛利率不修复或海外政策恶化。

通威股份（600438）同时涉及硅料和电池片，研究时要分开看：硅料看价格和成本，电池片看技术路线和开工率。不能简单用“光伏龙头”四个字概括。

福斯特（603806）、福莱特（601865）、石英股份（603688）、帝科股份（300842）代表辅材和耗材，研究重点是装机需求、竞争格局、原料成本和技术路线。辅材有时比主链更值得跟踪，因为部分环节竞争格局更好，但仍需用财报验证。

### 7. 小白误区

| 误区 | 正确做法 |
|---|---|
| 光伏长期空间大，所以每家公司都好 | 分主链、辅材、设备、电站看利润 |
| 价格下跌一定是坏事 | 对上游坏，对下游需求可能好 |
| 技术路线一炒就能兑现业绩 | 查量产、良率、订单和毛利率 |
| 只看装机增长 | 还要看价格和产能利用率 |

## 第六章：风电产业链

### 1. 概念解释

风电是把风能转化为电能，分陆上风电和海上风电。陆风更成熟，海风价值量更高、弹性更大，但审批、施工、并网和海工能力要求更高。

### 2. 产业链位置

| 环节 | 产品 | 研究重点 |
|---|---|---|
| 整机 | 风机整机 | 招标价格、交付、毛利率 |
| 叶片 | 风机叶片 | 大型化、材料成本 |
| 塔筒 | 支撑结构 | 海风项目、钢材成本 |
| 海缆 | 海上输电 | 海风审批、项目节奏、格局 |
| 轴承/铸件/主轴 | 核心零部件 | 国产替代、产能、客户 |
| 运营 | 风电场 | 利用小时、电价、资产收益率 |

### 3. 盈利模式

风电整机竞争激烈，价格下行会压制利润；零部件如果格局好、技术壁垒高、受益海风放量，可能比整机更有弹性。风电运营商则更像公用事业，看电价、利用小时和资产质量。

### 4. 行情轮动路径

风电行情常见路径：政策或海风审批改善 -> 风电招标恢复 -> 海缆、塔筒等零部件先反应 -> 整机交付改善 -> 运营商装机提升 -> 财报兑现。若只是招标量增加但招标价格持续下降，需要警惕整机利润承压。

### 5. 关键验证数据

| 数据 | 验证什么 |
|---|---|
| 风电招标量和价格 | 需求与竞争格局，需更新验证 |
| 海风核准和开工 | 海风项目是否落地，需更新验证 |
| 零部件订单 | 海缆、塔筒、轴承景气度，需更新验证 |
| 并网数据 | 交付是否兑现，需更新验证 |
| 钢材和铜价 | 成本压力，需更新验证 |

### 6. 公司案例

东方电缆（603606）代表海缆方向，研究重点是海风项目核准、海缆订单、交付节奏和毛利率，均需更新验证。受益情景是海风审批恢复和项目开工；证伪条件是海风项目延期、订单不及预期或竞争加剧。

大金重工（002487）、天顺风能（002531）代表塔筒和风电结构件，研究重点是海风订单、海外订单、钢材成本和产能布局，均需更新验证。

金风科技（002202）、明阳智能（601615）代表整机，研究重点是招标份额、交付、整机毛利率和风电运营资产，均需更新验证。

### 7. 小白误区

| 误区 | 正确理解 |
|---|---|
| 风电招标多就等于整机赚钱 | 招标价格下降可能压缩整机利润 |
| 海风消息就是业绩 | 需要核准、开工、交付、并网 |
| 所有零部件弹性一样 | 海缆、塔筒、轴承、铸件格局不同 |

## 第七章：电力设备与新型电力系统

### 1. 概念解释

新能源发电越多，电网压力越大。风光发电波动大，电力需要跨区域输送，配电网也要适应分布式光伏、储能、充电桩和数据中心用电。因此，电力设备是新能源后周期的重要方向。

### 2. 产业链位置

| 环节 | 产品 | 作用 |
|---|---|---|
| 特高压 | 换流阀、变压器、电抗器 | 远距离输电 |
| 主网设备 | 变压器、开关、GIS | 骨干电网建设 |
| 配网设备 | 配电变压器、开关柜、环网柜 | 城市和工商业配电 |
| 自动化 | 继保、调度、监控 | 提高电网智能化 |
| 电表 | 智能电表 | 计量和用电管理 |
| 虚拟电厂 | 软件平台 | 聚合储能、负荷和分布式资源 |

### 3. 盈利模式

电力设备公司主要靠电网招标订单赚钱。制造类公司看中标、交付、原材料成本和海外认证；软件和自动化公司看项目、系统和运维；虚拟电厂还要看电力市场规则是否成熟。

### 4. 行情轮动路径

电力设备行情常见路径：新能源装机提升 -> 消纳和并网压力增加 -> 政策强调电网投资 -> 特高压和主网先动 -> 配网、智能电表、自动化扩散 -> 虚拟电厂和电力信息化主题扩散 -> 订单和财报验证。

AI算力用电可能强化这一逻辑，但不能只看“AI用电”四个字，必须验证数据中心实际建设、变压器订单、配电设备订单和电网投资。

### 5. 关键验证数据

| 数据 | 验证什么 |
|---|---|
| 电网投资计划 | 行业景气度，需更新验证 |
| 国家电网/南网招标 | 订单节奏，需更新验证 |
| 特高压核准和开工 | 主网设备需求，需更新验证 |
| 配网改造政策 | 配网设备需求，需更新验证 |
| 变压器出口 | 海外电力设备需求，需更新验证 |
| 数据中心用电 | AI用电逻辑，需更新验证 |

### 6. 公司案例

国电南瑞（600406）代表电网自动化和电力信息化，研究重点是电网投资、调度系统、继保自动化和订单兑现，均需更新验证。它更偏确定性和系统能力，不是单纯题材股。

许继电气（000400）、平高电气（600312）、中国西电（601179）代表特高压和主网设备，研究重点是特高压核准、招标批次和中标金额，均需更新验证。

金盘科技（688676）、华明装备（002270）、思源电气（002028）代表变压器、分接开关、一次设备和出海方向，研究重点是海外订单、产能、毛利率和交付，均需更新验证。

### 7. 小白误区

| 误区 | 正确理解 |
|---|---|
| 新能源装机高就自动利好所有电网股 | 需要看具体招标和产品环节 |
| AI用电等于所有电力设备都涨 | 要验证数据中心建设和订单 |
| 电力设备没有周期 | 电网投资、原材料、交付也有周期 |

## 第八章：氢能产业链

### 1. 概念解释

氢能是用氢作为能源载体。它可以通过电解水制氢，也可以来自化工副产或化石能源制氢。绿色氢能想象空间大，但短期商业化慢，很多公司还处于示范项目、技术储备或小规模订单阶段。

### 2. 产业链位置

| 环节 | 产品 | 研究重点 |
|---|---|---|
| 制氢 | 电解槽、制氢系统 | 绿氢项目和设备订单 |
| 储运 | 储氢瓶、管道、液氢设备 | 安全、成本、标准 |
| 加氢 | 加氢站设备 | 建设数量和利用率 |
| 燃料电池 | 电堆、系统、膜电极、催化剂 | 车辆放量和核心材料 |
| 应用 | 交通、化工、钢铁、储能 | 商业模式和成本 |

### 3. 盈利模式

氢能公司短期多看设备订单和示范项目，长期看绿氢成本下降和应用场景放量。小白最容易犯的错误是把“远期空间大”当成“短期业绩确定”。氢能研究必须区分技术储备、示范项目、订单收入和利润兑现。

### 4. 行情轮动路径

氢能行情常见路径：政策规划 -> 示范城市或项目 -> 电解槽和设备先涨 -> 项目开工和中标验证 -> 燃料电池车或工业应用放量 -> 财报兑现。若没有订单和收入，只能按主题催化看待。

### 5. 关键验证数据

| 数据 | 验证什么 |
|---|---|
| 电解槽招标 | 制氢设备需求，需更新验证 |
| 绿氢项目开工 | 项目是否落地，需更新验证 |
| 燃料电池车销量 | 终端需求，需更新验证 |
| 加氢站利用率 | 商业化程度，需更新验证 |
| 氢能业务收入占比 | 公司是否真正受益，需更新验证 |

### 6. 公司案例

亿华通（688339）代表燃料电池系统，研究重点是燃料电池车放量、客户、收入和毛利率，均需更新验证。证伪条件是车辆推广不及预期、收入占比低或亏损扩大。

华电重工（601226）、东方电气（600875）、兰石重装（603169）涉及氢能设备或能源装备，研究时要分清主营业务和氢能业务占比，不能只看概念。

厚普股份（300471）、京城股份（600860）涉及加氢站和储氢装备，重点看订单、交付和收入占比，均需更新验证。

### 7. 小白误区

| 误区 | 正确理解 |
|---|---|
| 氢能空间大，所以短期股价一定强 | 主题和业绩要分开 |
| 有技术储备就等于能赚钱 | 必须看订单和收入 |
| 有合作协议就等于确定业绩 | 要看合同金额、交付和回款 |

## 第九章：新能源车与交叉方向

### 1. 新能源车

新能源车研究不能只看整车销量，还要看车型周期、价格战、单车毛利、零部件单车价值量和供应链地位。

| 环节 | 研究重点 | 代表公司 |
|---|---|---|
| 整车 | 销量、车型、毛利、库存 | 比亚迪、赛力斯、长安汽车、江淮汽车、北汽蓝谷 |
| 热管理 | 电动车热管理复杂度提升 | 三花智控、银轮股份 |
| 轻量化 | 铝合金、一体压铸、结构件 | 拓普集团、旭升集团、文灿股份、爱柯迪 |
| 智能化 | 座舱、域控、智能驾驶 | 德赛西威、华阳集团、均胜电子 |
| 充换电 | 补能基础设施 | 特锐德、盛弘股份、英可瑞 |

公司案例：比亚迪（002594）同时具有整车、电池和供应链优势，研究时要看销量、单车盈利、出口、车型结构和价格战影响，均需更新验证。德赛西威（002920）代表汽车电子和智能座舱，研究重点是定点项目、量产节奏和客户结构，均需更新验证。

### 2. 交叉方向

新能源交叉方向包括AI数据中心用电、液冷和热管理、数据中心储能、机器人电池、低空经济电池、电动船舶、工程机械电动化等。它们容易形成主题行情，但更需要验证收入占比。

研究路径是：先判断新场景是否真实增长，再判断公司是否有产品，再查是否有订单和客户，最后看收入占比和毛利率。没有收入占比的数据，一律标注“需更新验证”。

## 第十章：公司池与公司分析卡

### 1. 公司池使用原则

公司池不是推荐清单，而是观察对象。每家公司都要放回产业链位置里看：它赚资源钱、加工钱、设备钱、系统集成钱、运营钱，还是主题预期的钱？如果这个问题回答不清，先不要把它加入高优先级。

### 2. 重点公司分析卡

#### 宁德时代（300750）

#### 亿纬锂能（300014）

## 第十一章：案例教学：把小作文变成可验证假设

### 1. 从论坛发言到研究假设

假设论坛里有人说：“新能源这波先看锂电，锂价开始反弹，正极和电池排产也起来了。”这句话不能直接变成结论，要拆成四个假设：

| 原始说法 | 可验证假设 | 验证数据 |
|---|---|---|
| 锂价开始反弹 | 碳酸锂/氢氧化锂价格连续上涨 | 价格数据，需更新验证 |
| 正极排产起来 | 正极企业开工率提升 | 排产和行业调研，需更新验证 |
| 电池排产起来 | 电池厂出货和装机提升 | 装机、出货、公告，需更新验证 |
| 锂电先涨 | 板块内部资源、材料、电芯是否轮动 | 行情和成交，需更新验证 |

验证后可能出现三种结果：数据都支持，说明逻辑初步成立；只有锂价支持但排产不支持，可能只是资源反弹；只有股价涨但数据不支持，可能是情绪或补涨。

### 2. 从图片观点到产业链框架

如果图片只展示锂电周期传导，不能把它当成新能源完整框架。正确做法是把图片当成一个案例：它解释的是“锂电周期”，不是“新能源全部”。还要补充储能、光伏、风电、电力设备、氢能和新能源车。

图片观点的处理流程：

1. 识别它说的是哪个产业链。
2. 判断它只覆盖哪些环节。
3. 把遗漏环节补上。
4. 标出哪些是事实，哪些是推测。
5. 给每个推测配验证数据。

### 3. 从小作文到公告验证

| 小作文类型 | 常见说法 | 验证路径 | 证伪信号 |
|---|---|---|---|
| 订单型 | 某公司拿到大单 | 查中标公告、重大合同、客户公告 | 无公告、无合同、收入不体现 |
| 涨价型 | 产品涨价 | 查行业报价、调价函、毛利率 | 价格未涨或毛利率未改善 |
| 技术型 | 固态/钠电/钙钛矿突破 | 查专利、中试、客户认证、量产 | 只有概念无量产 |
| 客户型 | 进入大客户供应链 | 查定点公告、客户认证、车型销量 | 定点不量产或金额小 |
| 业绩型 | 中报大增 | 查业绩预告、财报、主营构成 | 一次性收益或现金流差 |
| 出海型 | 海外订单爆发 | 查出口、客户公告、认证、汇率 | 库存高、交付慢、贸易风险 |

## 第十二章：四个实战专题

### 1. 中报预增筛选

中报预增不是等公告出来才看，而是提前找可能改善的方向。筛选顺序是：行业数据改善 -> 公司订单改善 -> 毛利率修复 -> 低基数 -> 财务质量验证。

| 筛选线索 | 可能对应 | 验证方法 |
|---|---|---|
| 产品价格止跌 | 毛利率修复 | 价格和财报，需更新验证 |
| 排产提升 | 收入增长 | 排产、出货、订单，需更新验证 |
| 原料下降 | 成本改善 | 存货、采购成本，需更新验证 |
| 上年低基数 | 同比高增长 | 历史财报，需更新验证 |
| 合同负债增加 | 订单较好 | 财报科目，需更新验证 |
| 现金流改善 | 利润质量提升 | 经营现金流，需更新验证 |

警惕三类“假预增”：非经常性收益导致的增长、低基数但主业没改善、利润增长但经营现金流恶化。

### 2. 公告澄清判断

澄清公告不是简单利空。有些澄清会彻底证伪概念，有些只是说明热门业务占比小，但主业仍在改善。

| 澄清类型 | 判断 | 后续看什么 |
|---|---|---|
| 完全无相关业务 | 题材基本证伪 | 不再按该概念跟踪 |
| 有技术储备但未量产 | 只能当主题观察 | 中试、客户、量产 |
| 有业务但占比小 | 低优先级跟踪 | 收入占比是否提升 |
| 有订单但金额小 | 观察兑现 | 合同、交付、毛利 |
| 澄清热门概念，但主业改善 | 回到主业 | 主业订单和财务 |

### 3. 低位修复还是陷阱

低位股不是天然便宜。新能源很多公司从高位跌下来，可能是周期底部，也可能是盈利能力永久受损。

| 判断项 | 修复特征 | 陷阱特征 |
|---|---|---|
| 行业价格 | 止跌或企稳 | 继续下跌 |
| 毛利率 | 环比改善 | 继续恶化 |
| 库存 | 下降，减值减少 | 高库存，减值增加 |
| 现金流 | 经营现金流改善 | 应收和库存吞现金 |
| 订单 | 新订单或客户恢复 | 只有概念无订单 |
| 股价表现 | 龙头先稳，板块有序扩散 | 只有后排乱涨 |

### 4. 补涨末期识别

补涨不是不能研究，但补涨末期风险很高。识别补涨末期，要看龙头、后排、成交、公告和小作文质量。

| 信号 | 健康扩散 | 补涨末期 |
|---|---|---|
| 龙头 | 继续稳健上行 | 龙头滞涨或冲高回落 |
| 后排 | 有基本面线索 | 无收入占比也乱涨 |
| 成交 | 放量但承接好 | 高位巨量滞涨 |
| 公告 | 利好仍被认可 | 利好兑现后回落 |
| 小作文 | 能被公告验证 | 传闻越来越多但验证少 |
| 情绪 | 分歧后修复 | 亏钱效应扩大 |

## 第十三章：每日复盘流程

每天复盘新能源，不是简单看谁涨停，而是回答六个问题：

1. 今天最强的是哪个细分方向？
2. 它上涨的原因是政策、价格、订单、业绩、技术还是情绪？
3. 龙头是否领涨？后排是否补涨？
4. 有没有公告、财报、价格或招投标数据验证？
5. 哪些信息只是小作文，仍需更新验证？
6. 如果明天要继续跟踪，最关键的数据是什么？
