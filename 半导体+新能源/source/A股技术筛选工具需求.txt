三、核心筛选逻辑

请参考以下交易纪律进行筛选。

重点不是预测股价，而是判断：

* 当前趋势是否健康；
* 是否处于主升初期或主升中段；
* 是否缩量回调；
* 是否出现长下影止跌；
* 是否存在明显供应进场；
* 是否已经出现过启动信号；
* 当前价格位置是否过高；
* 波动风险是否过大；
* 是否适合进入重点观察池；
* 是否需要等待次日确认。

六、量价质量判断

请实现 analyze_volume_price(df) 函数。

判断内容：

1. 最近 1-3 日是否缩量回调；
2. 是否出现长下影；
3. 上涨是否放量；
4. 回调是否缩量；
5. 是否出现放量大阴线；
6. 是否存在明显供应进场。

量化参考：

缩量回调：

* 最近 1-3 日中存在收跌；
* 成交量低于前 5 日均量的 80%；
* 没有跌破 MA20。

长下影：

* 下影线长度 / 当日振幅 > 40%。

供应进场：

* 当日跌幅大于 4%；
* 成交量大于前 5 日均量的 1.5 倍；
* 收盘价靠近最低价；
* 或连续出现放量阴线。

输出示例：

```python
{
    "volume_score": 90,
    "volume_status": "量价健康",
    "comment": "缩量回调，长下影止跌，暂未看到明显供应进场"
}
```

八、位置判断

请实现 analyze_position(df) 函数。

计算当前价格在近 120 日区间的位置：

```python
position_percentile = (当前收盘价 - 120日最低价) / (120日最高价 - 120日最低价)
```

分层：

* 0% - 30%：低位；
* 30% - 60%：中位；
* 60% - 80%：偏高但可接受；
* 80% - 90%：高位，谨慎追高；
* 90% - 100%：明显高位，不适合追高。

输出示例：

```python

    "position_status": "偏高但可接受",
    "position_score": 70,

}
```

十、综合评分

请实现 calculate_score(result) 函数。

总分 100 分，权重如下：

* 趋势结构：30 分；
* 量价质量：25 分；
* 启动信号：15 分；
* 位置安全：15 分；
* 风险可控：15 分。

评级标准：

* 85 分以上：重点观察；
* 75 - 84 分：等待买点；
* 65 - 74 分：仅跟踪；
* 65 分以下：暂时剔除。

强制降级规则：

1. 放量跌破 MA20：降为“暂时剔除”；
2. 明显供应进场：最多只能评为“仅跟踪”；
3. 位置超过 90% 且 ATR 超过 9%：最多只能评为“等待买点”；
4. 数据不足：标记为“数据不足”，不参与高分排序；
5. 最近 20 日平均成交额过低：降级。
