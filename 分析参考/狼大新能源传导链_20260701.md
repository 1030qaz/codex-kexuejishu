# 狼大新能源传导链补充（2026-07-01）

> 仅用于论坛发言学习和交易复盘，不构成投资建议。涉及具体标的时，只作观察清单和验证框架，不输出买卖指令。

## 口径修正

这里讨论的是狼大历史发言中的“新能源传导链”，不是把新能源简化成“铜箔传导链”。铜箔只是材料端或双属性样本之一，不能代替电池、储能、业绩、资金外溢和量价位置这条完整链。

## 传导链拆解

| 链条层级 | 狼大视角拆解 | 复盘要点 |
| --- | --- | --- |
| 1. 资金起点 | 科技主线吃到利润后，部分资金外溢寻找低位、业绩可验证方向。 | 新能源不是替代科技，而是科技高位后承接部分利润的方向。 |
| 2. 触发信号 | 亿纬锂能业绩显示储能加速，且利润没有被上游长协明显侵蚀。 | 先看业绩确认，再找产业链映射，不能先有票再找理由。 |
| 3. 产业传导 | AIDC/高波动负荷 -> 储能扩容 -> 电池排产/中报 -> 材料顺价/涨价 -> 供应商利润验证。 | 铜箔、隔膜、电解液、结构件、PCS 都只是不同环节。 |
| 4. 位置口径 | 低位看逻辑，高位看量价；相对低位、支撑位、缩量洗刷刷更符合狼大口径。 | 高位加速票不能再只用“储能/涨价/国产替代”解释。 |
| 5. 风险口径 | 板块太明牌、自媒体太热、散户多、宁德小作文、锂矿供给变量都会增加波动。 | 新能源即使逻辑对，也可能先洗；仓位和失效条件比口号重要。 |

## 后续复盘模板

先问“这条发言落在链条哪一层”，再问“是否有业绩/订单/价格/量价验证”，最后问“位置是低位逻辑还是高位量价”。如果三项缺一项，只能标观察，不强行脑补。

当前优先级：先盯亿纬锂能这类业绩锚和储能电池链，再看中一科技等历史样本是否在支撑位缩量承接；科达利、星源、鹏辉、盛弘作为传导链不同环节观察；铜箔类只作材料端分支验证。
